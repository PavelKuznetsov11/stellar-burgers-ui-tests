# Stellar Burgers UI Tests

Проект с UI-автотестами для web-приложения Stellar Burgers.

Цель проекта — проверить основные пользовательские сценарии через интерфейс приложения: авторизацию, переходы между страницами, личный кабинет и оформление заказа.

## Что тестируется

- авторизация пользователя
- переход в личный кабинет
- выход из аккаунта
- переходы между разделами приложения
- оформление заказа
- отображение элементов интерфейса
- работа основных пользовательских сценариев

## Технологии

- Python
- Selenium WebDriver
- Pytest
- Allure
- Page Object Model
- Git

## Что проверяют тесты

- корректность работы UI
- отображение важных элементов интерфейса
- переходы между страницами
- сценарии авторизации
- работу личного кабинета
- работу формы оформления заказа
- стабильность ключевых пользовательских сценариев

Page Object Model
В проекте используется паттерн Page Object Model.
Логика работы со страницами вынесена в отдельные классы, а локаторы — в отдельные файлы.
Такой подход упрощает поддержку автотестов: при изменении интерфейса достаточно обновить локатор или метод страницы, а не переписывать все тесты.

Как запустить проект

Клонировать репозиторий:
git clone https://github.com/PavelKuznetsov11/stellar-burgers-ui-tests.git

Перейти в папку проекта:
cd stellar-burgers-ui-tests

Создать и активировать виртуальное окружение:
python -m venv venv

Для Windows:
venv\Scripts\activate

Для macOS / Linux:
source venv/bin/activate

Установить зависимости:
pip install -r requirements.txt

Запустить тесты:
pytest -v

Запустить тесты с генерацией Allure-результатов:
pytest --alluredir=allure_results

Открыть Allure-отчёт:
allure serve allure_results
